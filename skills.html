psychology ソフトスキル

強みとなるスキル 任意
技術以外の強みをまとめています。

コミュニケーション力: 相手に伝わる言葉を意識しながら、画面構成や説明の分かりやすさを大切にして制作を進めています。

問題解決力: エラーログ、画面の挙動、コードの流れを一つずつ確認し、原因を切り分けながら修正する姿勢を身につけています。

継続学習力: 分からないことをそのままにせず、調査・試作・修正を繰り返しながら少しずつ形にしていくことを意識しています。

trending_up 今後学びたい技術

学習予定 任意

REST API 設計やフロントエンドとバックエンドの連携を、より実践的に理解できるように学習を深めたいと考えています。

データベース設計とSQL運用に強くなり、開発環境だけでなく本番運用を意識した設計ができるようになりたいです。

Spring Security を使った認証・認可の理解を深め、ログイン機能や権限制御をより安全に実装できるようにしたいです。

Git を使ったチーム開発の進め方やレビューを意識しながら、複数人開発でも安定して進められる力を伸ばしたいです。
